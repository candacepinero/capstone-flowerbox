# Flower Box
This Flower Box project was created using material UI and React. Using Material UI's collection of prebuilt components, and MUI's mobile-first design, help create a well styled responses web application.

## Technologies used
* React 
* Material UI 
* React Router Dom6
* Netlify
* HTML
* CSS
